# Development

## Howto build
* Install QTCreator [IDE](https://qt-project.org/downloads)
* Clone source code from [here](https://github.com/eguzki/penyamanager)
* From QTCreator import **penyamanager.pro** file
* Build: CTRL + SHIFT + B
* Run: CTRL + R

## Developed using:
* QT Creator 3.0.1
* QT 5.2.1
* GCC 4.8.2, 64 bit
